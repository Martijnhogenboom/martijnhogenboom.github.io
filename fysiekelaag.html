Wat is de fysieke laag in het 3 lagenmodel?

We gaan onderzoeken wat de fysieke laag is in het 3 lagenmodel aan de hand van de processor, kloksnelheid, verwerkingssnelheid en het geheugen.

De processor

Wat is een processor? De Processor wordt ook wel de CPU genoemd. CPU staat voor central Processing unit. De CPU is een deel in je telefoon die alles van je telefoon aanstuurt en goed laat functioneren. Hoe werkt een CPU? Als je een app opent op je telefoon dan zorgt de CPU ervoor dat die dit opslaat in het geheugen van een telefoon (ophaalfase). Daarna wordt dit omgezet naar binaire getallen (decodeerfase). dan stuurt de CPU de binaire getallen uit en dan opent de app op jouw telefoon (uitvoerfase) en dan gebeurt dit elke keer opnieuw.

De Kloksnelheid

De Kloksnelheid van een telefoon is de snelheid waarmee de CPU van je telefoon werkt. De kloksnelheid wordt vaak uitgedrukt in gigahertz (GHz). Dit geeft aan hoeveel bewerkingen de CPU per seconde kan uitvoeren. Een hogere kloksnelheid betekent dat de CPU in principe sneller kan werken, wat leidt tot betere prestaties bij het uitvoeren van taken zoals apps openen en gamen. Maar de kloksnelheid is niet de enige factor die bepaald hoe snel een telefoon werkt.
