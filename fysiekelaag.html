Wat is de fysieke laag in het 3 lagenmodel?

We gaan onderzoeken wat de fysieke laag is in het 3 lagenmodel aan de hand van de processor, kloksnelheid, verwerkingssnelheid en het geheugen.

De processor

Wat is een processor? De Processor wordt ook wel de CPU genoemd. CPU staat voor central Processing unit. De CPU is een deel in je telefoon die alles van je telefoon aanstuurt en goed laat functioneren. Hoe werkt een CPU? Als je een app opent op je telefoon dan zorgt de CPU ervoor dat die dit opslaat in het geheugen van een telefoon (ophaalfase). Daarna wordt dit omgezet naar binaire getallen (decodeerfase). dan stuurt de CPU de binaire getallen uit en dan opent de app op jouw telefoon (uitvoerfase) en dan gebeurt dit elke keer opnieuw.

De Kloksnelheid

De Kloksnelheid van een telefoon is de snelheid waarmee de CPU van je telefoon werkt. De kloksnelheid wordt vaak uitgedrukt in gigahertz (GHz). Dit geeft aan hoeveel bewerkingen de CPU per seconde kan uitvoeren. Een hogere kloksnelheid betekent dat de CPU in principe sneller kan werken, wat leidt tot betere prestaties bij het uitvoeren van taken zoals apps openen en gamen. Maar de kloksnelheid is niet de enige factor die bepaald hoe snel een telefoon werkt.

Het geheugen

Het geheugen van een telefoon bestaat uit verschillende opslag typen. De twee belangrijkste vormen van geheugen zijn de RAM en de ROM. De RAM is een tijdelijk geheugen en de ROM is een permanent geheugen.

De RAM

De RAM staat voor Random access memory. De RAM is een type geheugen dat tijdelijke gegevens opslaat die de processor op dat moment nodig heeft om taken snel uit te voeren. De RAM zorgt ervoor dat een telefoon meerdere taken tegelijk kan uitvoeren (multitasking) en zorgt voor snelle toegang tot actieve apps en gegevens. Bijvoorbeeld, wanneer je schakelt tussen apps, slaat de telefoon de status van deze apps in het RAM op zodat je snel verder kunt gaan.

De ROM

ROM staat voor Read Only memory. De ROM verwijst meestal naar het interne opslaggeheugen van een telefoon, waar besturingssystemen, apps, foto’s, video’s en andere bestanden permanent worden opgeslagen. De term Read-Only verwijst naar het oorspronkelijke ontwerp, waarin gegevens alleen gelezen en niet gewist konden worden. ROM slaat belangrijke bestanden en het besturingssysteem van de telefoon op. Dit geheugen blijft behouden, zelfs wanneer de telefoon uitgeschakeld is. Het bevat gegevens die de gebruiker nodig heeft voor de dagelijkse functies van de telefoon, zoals foto’s, apps en bestanden.
