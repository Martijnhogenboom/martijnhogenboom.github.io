Wat is de fysieke laag in het 3 lagenmodel?

We gaan onderzoeken wat de fysieke laag is in het 3 lagenmodel aan de hand van de processor, kloksnelheid, verwerkingssnelheid en het geheugen.

De processor

Wat is een processor? De Processor wordt ook wel de CPU genoemd. CPU staat voor central Processing unit. De CPU is een deel in je telefoon die alles van je telefoon aanstuurt en goed laat functioneren.

De Kloksnelheid

De Kloksnelheid van een telefoon is de snelheid waarmee de CPU van je telefoon werkt. De kloksnelheid wordt vaak uitgedrukt in gigahertz (GHz). Dit geeft aan hoeveel bewerkingen de CPU per seconde kan uitvoeren. Een hogere kloksnelheid betekent dat de CPU in principe sneller kan werken, wat leidt tot betere prestaties bij het uitvoeren van taken zoals apps openen en gamen.

Het geheugen

Het geheugen van een telefoon bestaat uit verschillende opslag typen. De twee belangrijkste vormen van geheugen zijn de RAM en de ROM. De RAM is een tijdelijk geheugen en de ROM is een permanent geheugen.

De RAM

De RAM staat voor Random access memory. De RAM is een type geheugen dat tijdelijke gegevens opslaat die de processor op dat moment nodig heeft om taken snel uit te voeren. De RAM zorgt ervoor dat een telefoon meerdere taken tegelijk kan uitvoeren (multitasking) en zorgt voor snelle toegang tot actieve apps en gegevens.

De ROM

De ROM staat voor Read-Only Memory en is een type geheugen waarin permanente gegevens worden opgeslagen. Alles wat in de ROM komt wordt permanent opgeslagen.
